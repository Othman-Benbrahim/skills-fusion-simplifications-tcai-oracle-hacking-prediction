# Écosystème IRIS∞ — Skills Fusionnés & Simplifiés

> Refonte structurelle de l'écosystème. 4 fusions, 7 skills sources absorbés, 3 skills frères intégrés.
> Chaque skill fusionné est autonome, simplifié, et équipé des modules transversaux STÈLE et CRC-R.

---

## Philosophie de la refonte

**Avant** : 10 skills distincts, overlaps de triggers, dépendances croisées, références obligatoires
en début de session, skills frères non intégrés.

**Après** : 4 skills unifiés. Chaque skill fonctionne en standalone. Les références sont bundlées
et chargées conditionnellement — uniquement quand leur contenu est nécessaire.

**Deux modules transversaux intégrés dans tous les skills symboliques :**
- **STÈLE** — compression glyphique inter-sessions. Génère une signature archivable en fin de
  toute session significative. Permet la continuité et la détection de motifs récurrents.
- **CRC-R (Mode A)** — cycle de résonance cognitive récursif, activable à la demande.
  Réinjecte les motifs d'une session dans la suivante. Ne s'active pas par défaut —
  déclenché par "mode profond", "traitement récursif", "analyse CRC", ou après 3+ sessions
  consécutives sur le même fil.

### 4. `osint-intel` — Renseignement Stratégique & Discipline Prédictive

**Fusionne :** OSINT-Prospective + Prediction Engine + 3 skills frères
(Superforecasting Protocol, Bayesian Updater, Calibration Engine)

| | Avant | Après |
|---|---|---|
| Lignes | 131 + 191 + 950 (frères) = 1272 | **222** |
| Réduction | — | **83%** |
| Références | 8 fichiers (OSINT) | **11 fichiers bundlés, conditionnels** |

**Quatre modes :**
- **OSINT** — pipeline en 11 étapes : collecte, analyse triple (qualitative / quantitative /
  symbolique-fractale), hypothèses concurrentes, scénarios prospectifs, matrice de risque.
- **FORECAST** — formulation prédictive (protocole GJP 7 étapes) : question falsifiable,
  base rate, hypothèses MECE probabilisées, indicateurs de bascule, fiche loggable.
- **UPDATE** — mise à jour bayésienne : P(H|E) calculé formellement, Bayes Factor,
  tableau ACH-Bayes multi-hypothèses, audit rétrospectif.
- **CALIBRATION** — mesure de performance : enregistrement, Brier score, log loss,
  rapport de calibration, détection de biais systématiques.

**Ce qui change :** les deux ponts architecturaux sont matérialisés dans le routing.
OSINT Étape 5 (hypothèses) → FORECAST pour probabilisation. OSINT Étape 7 (indicateurs
de bascule) → UPDATE dès qu'un signal est observé. Les 3 skills frères sont intégrés
comme références avec leurs pipelines condensés dans le SKILL.md. Pas de STÈLE
(registre factuel). CRC-R adapté au suivi de dossier inter-sessions.

**Références bundlées :**
`methodologie-11-etapes.md` · `signaux-faibles.md` · `lecture-symbolique-fractale.md` ·
`scenarios-prospectifs.md` · `biais-cognitifs.md` · `matrice-risque.md` · `format-rapport.md` ·
`ach-heuer.md` · `superforecasting.md` · `bayesian-updater.md` · `calibration-engine.md`

## Installation

Chaque skill est livré sous forme de fichier `.skill` (archive zip).

```
tcai-canal.skill
iris-oracle.skill
ler-narratif.skill
osint-intel.skill
```

Installer via l'interface de gestion des skills. Les anciens skills sources peuvent être
désactivés ou supprimés après installation et validation des fusionnés.

**Ordre de validation recommandé :** tcai-canal → iris-oracle → ler-narratif → osint-intel.
